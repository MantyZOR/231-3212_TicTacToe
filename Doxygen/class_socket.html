Socket::Socket ( )

Конструктор класса Socket.

Создает сокет с использованием протокола IPv4, типа SOCK_STREAM (TCP) и устанавливает адрес и порт для прослушивания входящих соединений.

Исключения
std::system_errorв случае ошибки при создании сокета.

Создает сокет с использованием протокола IPv4 (AF_INET), типа SOCK_STREAM (TCP) и устанавливает адрес и порт для прослушивания входящих соединений (по умолчанию - порт 8083).
