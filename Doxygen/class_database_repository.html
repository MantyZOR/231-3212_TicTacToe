Методы

◆ connectToDatabase()

void DatabaseRepository::connectToDatabase ( )

Устанавливает соединение с базой данных SQLite.

Проверяет доступность драйвера SQLite, создает файл базы данных, если он не существует, и пытается установить соединение.

Выводит сообщение об ошибке в консоль, если не удалось установить соединение с базой данных.

◆ createTables()

void DatabaseRepository::createTables ( )

Создает таблицы в базе данных, если они не существуют.

Создает таблицы "Users" (пользователи) и "Games" (игры), если они еще не созданы.
